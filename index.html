Habilidades


  • HTML

    Tengo un sólido dominio de HTML5, lo que me permite estructurar páginas web de manera clara, semántica y accesible. Cuido cada detalle del marcado para asegurar que el contenido sea entendible tanto para los usuarios como para los motores de búsqueda, garantizando bases sólidas para cualquier proyecto web.


  • CSS

    Me encanta trabajar con CSS, porque me permite dar estilo, color y vida a las páginas web. Experimento con tipografías, paletas modernas y efectos dinámicos para que cada proyecto tenga personalidad propia.

    Manejo React.js para desarrollar interfaces de usuario modernas, modulares y eficientes. Trabajo con componentes reutilizables, hooks, manejo de estados y consumo de APIs, lo que me permite crear aplicaciones dinámicas y escalables.

    Con JavaScript transformo páginas estáticas en experiencias interactivas, logrando que el usuario no solo navegue, sino que también interactúe con el contenido.
